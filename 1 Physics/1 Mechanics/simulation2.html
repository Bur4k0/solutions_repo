Investigating the Dynamics of a Forced Damped Pendulum

Motivation

The forced damped pendulum is a captivating example of a physical system with intricate behavior resulting from the interplay of damping, restoring forces, and external driving forces. By introducing both damping and external periodic forcing, the system transitions from simple harmonic motion to a range of behaviors, including resonance, chaos, and quasiperiodic motion.

Python Simulation Example

Below is an example of a Python script using the Runge-Kutta method to simulate the forced damped pendulum.

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

# Parameters
b = 0.2   # Damping coefficient
A = 1.2   # Driving force amplitude
omega = 2.0  # Driving frequency
g = 9.81  # Gravity
L = 1.0   # Pendulum length

# Differential equations
def pendulum(t, y):
    theta, omega_t = y
    dtheta_dt = omega_t
    domega_dt = -b * omega_t - np.sin(theta) + A * np.cos(omega * t)
    return [dtheta_dt, domega_dt]

# Time span
t_span = (0, 50)
t_eval = np.linspace(0, 50, 1000)

# Initial conditions
y0 = [np.pi/4, 0]  # Small initial angle

# Solve using Runge-Kutta method
sol = solve_ivp(pendulum, t_span, y0, t_eval=t_eval)

# Plot results
plt.figure(figsize=(8,5))
plt.plot(sol.t, sol.y[0], label="Theta (Angle)")
plt.xlabel("Time (s)")
plt.ylabel("Angle (radians)")
plt.title("Forced Damped Pendulum Simulation")
plt.legend()
plt.grid()
plt.show()

This script solves the motion equations and plots the angular displacement over time. The behavior can be analyzed further by varying b, A, and omega.
