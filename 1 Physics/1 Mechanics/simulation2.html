Investigating the Dynamics of a Forced Damped Pendulum

1. Theoretical Foundation

The forced damped pendulum is governed by a second-order nonlinear differential equation that accounts for gravitational restoring force, damping, and an external periodic driving force. The equation of motion is:

\(\ddot{\theta} + 2\beta\dot{\theta} + \omega_0^2 \sin(\theta) = F \cos(\omega t)\)

Where:

  • \(\theta\): angular displacement (radians)
  • \(\dot{\theta}\): angular velocity
  • \(\ddot{\theta}\): angular acceleration
  • \(\beta = b/(2m)\): damping coefficient (\(b\) is damping constant, \(m\) is mass)
  • \(\omega_0 = \sqrt{g/L}\): natural frequency (\(g\) is gravity, \(L\) is pendulum length)
  • \(F\): amplitude of the external driving force (normalized by \(mL\))
  • \(\omega\): driving frequency
  • \(t\): time

Small-Angle Approximation

For small angles (\(\theta \ll 1\)), \(\sin(\theta) \approx \theta\), simplifying the equation to:

\(\ddot{\theta} + 2\beta\dot{\theta} + \omega_0^2 \theta = F \cos(\omega t)\)

The general solution is a sum of a homogeneous (transient) and particular (steady-state) solution:

  • Homogeneous: \(\theta_h(t) = e^{-\beta t} [A \cos(\omega_d t) + B \sin(\omega_d t)]\), where \(\omega_d = \sqrt{\omega_0^2 - \beta^2}\).
  • Particular: \(\theta_p(t) = C \cos(\omega t) + D \sin(\omega t)\), with:
    \(C = \frac{F (\omega_0^2 - \omega^2)}{(\omega_0^2 - \omega^2)^2 + (2\beta\omega)^2}, \quad D = \frac{F (2\beta\omega)}{(\omega_0^2 - \omega^2)^2 + (2\beta\omega)^2}\)
    Amplitude: \(A = \sqrt{C^2 + D^2} = \frac{F}{\sqrt{(\omega_0^2 - \omega^2)^2 + (2\beta\omega)^2}}\).

Resonance Conditions

Resonance occurs when \(\omega \approx \omega_0\), maximizing amplitude near \(\omega = \sqrt{\omega_0^2 - 2\beta^2}\) (for \(\beta < \omega_0/\sqrt{2}\)). In the nonlinear case, resonance can lead to complex dynamics like chaos.

2. Analysis of Dynamics

Influence of Parameters

  • Damping (\(\beta\)): Higher \(\beta\) reduces amplitude; overdamped (\(\beta > \omega_0\)), critically damped (\(\beta = \omega_0\)), or underdamped (\(\beta < \omega_0\)).
  • Driving Amplitude (\(F\)): Larger \(F\) increases amplitude and may induce chaos.
  • Driving Frequency (\(\omega\)): Near \(\omega_0\), resonance; otherwise, quasiperiodic or chaotic motion.

Transition to Chaos

In the nonlinear regime, the system exhibits periodic, quasiperiodic, or chaotic motion, observable via bifurcations (e.g., period-doubling).

3. Practical Applications

  • Energy Harvesting: Piezoelectric devices convert vibrations to electricity.
  • Suspension Bridges: Periodic forces (wind) can cause resonance or chaos.
  • Oscillating Circuits: Driven RLC circuits mirror the pendulum’s dynamics.

4. Implementation

Below is a Python script using the Runge-Kutta method to simulate the pendulum and generate visualizations. 

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import odeint

# Parameters
g = 9.81  # gravity (m/s^2)
L = 1.0   # pendulum length (m)
omega0 = np.sqrt(g / L)  # natural frequency
beta = 0.1  # damping coefficient
F = 1.2     # driving amplitude
omega = 1.4 * omega0  # driving frequency

# Differential equation
def pendulum_deriv(state, t, beta, omega0, F, omega):
    theta, theta_dot = state
    dtheta_dt = theta_dot
    dtheta_dot_dt = -2 * beta * theta_dot - omega0**2 * np.sin(theta) + F * np.cos(omega * t)
    return [dtheta_dt, dtheta_dot_dt]

# Time array
t = np.linspace(0, 50, 10000)

# Initial conditions
theta0 = 0.1  # radians
theta_dot0 = 0.0
state0 = [theta0, theta_dot0]

# Solve ODE
solution = odeint(pendulum_deriv, state0, t, args=(beta, omega0, F, omega))
theta, theta_dot = solution.T

# Phase portrait
plt.figure(figsize=(10, 6))
plt.plot(theta, theta_dot, lw=0.5)
plt.title("Phase Portrait")
plt.xlabel(r"$\theta$ (rad)")
plt.ylabel(r"$\dot{\theta}$ (rad/s)")
plt.grid(True)

# Poincaré section
T = 2 * np.pi / omega
poincare_idx = np.arange(0, len(t), int(T / (t[1] - t[0])))
plt.figure(figsize=(10, 6))
plt.plot(theta[poincare_idx], theta_dot[poincare_idx], 'o', ms=2)
plt.title("Poincaré Section")
plt.xlabel(r"$\theta$ (rad)")
plt.ylabel(r"$\dot{\theta}$ (rad/s)")
plt.grid(True)

# Bifurcation diagram
F_values = np.linspace(0.5, 1.5, 100)
theta_max = []
for F in F_values:
    sol = odeint(pendulum_deriv, state0, t, args=(beta, omega0, F, omega))
    theta_max.append(np.max(sol[-1000:, 0]))
plt.figure(figsize=(10, 6))
plt.plot(F_values, theta_max, 'k.', ms=1)
plt.title("Bifurcation Diagram")
plt.xlabel("Driving Amplitude (F)")
plt.ylabel(r"Max $\theta$ (rad)")
plt.grid(True)

plt.show()

Visualizations

Phase Portrait: \(\theta\) vs. \(\dot{\theta}\), showing periodic loops or chaotic scattering.

Poincaré Section: Points sampled at driving period, indicating periodic, quasiperiodic, or chaotic behavior.

Bifurcation Diagram: Max \(\theta\) vs. \(F\), revealing transitions to chaos.

Discussion and Limitations

Limitations: Assumes constant \(\beta\), \(F\), and \(\omega\).
Extensions: Nonlinear damping or stochastic forcing could enhance realism.
