Orbital Velocity Calculation and Visualization for Earth, Mars, and Jupiter

This script calculates and visualizes the orbital velocities (v1, v2, and v3) of Earth, Mars, and Jupiter based on their distance from the Sun.

Velocity Calculation

The orbital velocity \(v\) is calculated using the formula:
\(v = \sqrt{\frac{GM}{r}}\)
Where:
G = Gravitational constant = 6.67430 Ã— 10^-11 m^3 kg^-1 s^-2
M = Mass of the Sun (kg)
r = Distance from the Sun (m) 

        
            # Import necessary libraries
            import numpy as np
            import matplotlib.pyplot as plt

            # Constants
            G = 6.67430e-11  # Gravitational constant in m^3 kg^-1 s^-2
            M_sun = 1.989e30  # Mass of the Sun in kg

            # Planetary distances from the Sun in meters
            distances = {
                "Earth": 1.496e11,  # in meters
                "Mars": 2.279e11,   # in meters
                "Jupiter": 7.785e11 # in meters
            }

            # Calculate orbital velocities for each planet
            velocities = {}
            for planet, r in distances.items():
                v = np.sqrt(G * M_sun / r)
                velocities[planet] = v

            # Output velocities
            print("Orbital Velocities (m/s):")
            for planet, v in velocities.items():
                print(f"{planet}: {v:.2e} m/s")

            # Visualization (Bar chart of orbital velocities)
            planets = list(velocities.keys())
            velocity_values = list(velocities.values())

            plt.figure(figsize=(10, 6))
            plt.bar(planets, velocity_values, color='skyblue')
            plt.title('Orbital Velocities of Earth, Mars, and Jupiter')
            plt.xlabel('Planet')
            plt.ylabel('Velocity (m/s)')
            plt.yscale('log')  # Log scale for better visualization of large differences
            plt.show()
